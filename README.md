## Endpoints:

### Get:

```http request
GET http://localhost:5000/read/
```

### Post:

```http request
POST http://localhost:5000/create/
```

### Update:

```http request
PATCH http://localhost:5000/update/
```

### Delete:

```http request
DELETE http://localhost:5000/delete/(id)
```
